## 强制阻塞项（任一命中不得标「满配」）

- 权威源没有的 schema/返回码写成已冻结
- 待补 API 写入正式接口表
- 缺字段展示规则或错误示例
- 用「同 xx.md」代替独立字段定义
- 租户级路径写成 BOSS 平台正式契约
- PRD 与详文页面职责明显不一致

## 四轮回审

1. 权威源对照（v1.yaml + services/v1.yaml）
2. 跨材料一致（详文 > SPEC > PRD）
3. 反自造契约
4. 页面能力降噪
